================================================================
  OVERVIEW
================================================================

This project implements a complete neural cryptanalysis framework
that trains machine learning models to distinguish reduced-round
block ciphers from random permutations. Given a pair of ciphertexts
(C, C') produced from plaintexts with a fixed XOR difference ΔP,
the models learn to predict whether the outputs came from a real
cipher or a random permutation.

Approach inspired by Gohr (2019) — differential neural cryptanalysis.

================================================================
  PROBLEM STATEMENT
================================================================

  Let F be an unknown function — either:
    (a) A reduced-round block cipher E_k, or
    (b) A uniformly random permutation π

  Given input pairs with a fixed XOR difference:
      (P, P ⊕ ΔP)  →  (C, C') = (F(P), F(P ⊕ ΔP))

  Goal: Determine whether F is a real cipher or random permutation.

  Key insight: Cipher outputs exhibit structured XOR differences
  due to the algebraic structure of the round function, while
  random permutations produce uniform noise. This structure is
  learnable by neural networks.

================================================================
  DATASET GENERATION
================================================================

  1. Sample random plaintext P using os.urandom
  2. Compute P' = P ⊕ ΔP  (fixed Gohr-style difference)
  3. Encrypt both: (C, C') = (E_k(P), E_k(P'))
  4. Label: y=1 (cipher) or y=0 (random permutation)
  5. Apply chosen representation transform to (C, C')

  Fixed input differences (ΔP):
    simon32   : 0x40000000
    gift64    : 0x0000000000000001
    skinny64  : 0x0000000000000001
    craft     : 0x0000000000000001
    present   : 0x0000000000000001
    gift128   : 0x...0001  (128-bit)
    skinny128 : 0x...0001  (128-bit)
    pyjamask  : 0x...0001  (96-bit)

================================================================
  INPUT REPRESENTATIONS — 7 TOTAL
================================================================

  Representation   Shape              Accuracy  Notes
  --------------   -----              --------  -----
  delta            (block_bits,)      1.0000    bits(C XOR C') — best signal
  statistical      (59,)              0.9983    HW+nibble+XOR bits+autocorr
  joint            (6*block_bits,)    0.9933    P,P',C,C',ΔP,ΔC — white-box
  concat           (2*block_bits,)    0.9858    bits(C) || bits(C')
  raw              (2*block_bits,)    0.9758    Same as concat, float32 dtype
  word             (3*n_words,)       0.9708    16-bit word-level normalised
  bitslice         (2*block_bits,)    0.9592    Interleaved bits — CNN-friendly

  Statistical representation (59 features for simon32):
    Group 1 (7)  : HW(C), HW(C'), HW(ΔC), |ΔHW|, bit corr, HW_hi, HW_lo
    Group 2 (8)  : Per-nibble HW of ΔC
    Group 3 (32) : Raw XOR bit vector bits(C XOR C')
    Group 4 (4)  : Per-byte HW of ΔC
    Group 5 (8)  : Autocorrelation of ΔC bits at lags 1-8

================================================================
  MACHINE LEARNING MODELS — 4 TOTAL
================================================================

  1. MLP — Multi-Layer Perceptron
     Architecture : Linear(1024)->LN->GELU x5 + 2 skip -> Linear(1)
     Params       : 753,537 (delta) / 804,737 (concat)
     Best rep     : delta
     Best accuracy: 1.0000  (r=4, delta, N=4000, 15 epochs)

  2. CNN — Convolutional Neural Network
     Architecture : 3-ch (C,C',XOR) -> MultiScale(k=1,3,5) ->
                    3 ResStages -> global avg+max+std -> MLP head
     Params       : 313,141
     Best rep     : concat / bitslice
     Best accuracy: 0.9762  (r=4, concat, N=8000, 10 epochs)

  3. Siamese Network
     Architecture : Shared encoder (2L) -> concat + |diff| -> classifier
     Params       : 24,897
     Best rep     : concat
     Best accuracy: 0.9869  (r=4, concat, N=8000, 15 epochs)

  4. MINE — Mutual Information Neural Estimator
     Architecture : Augmented [C||C'||ΔC] -> Linear(512)->LN->GELU x4
                    + 2 skip connections -> Linear(1)
     Params       : 540,929
     Best rep     : concat (XOR channel added internally)
     Best accuracy: 0.9969  (r=4, concat, N=8000, 15 epochs)
     Reference    : Belghazi et al., ICML 2018

================================================================
  HOW TO RUN THE PROJECT
================================================================

  All commands must be run from inside AC_Project/

  STEP 1 — Install dependencies
  --------------------------------
      pip install torch numpy tqdm matplotlib

  STEP 2 — Verify all ciphers
  --------------------------------
      python neural_cryptanalysis/ciphers/verify_all_ciphers.py

  Runs test vectors, avalanche checks, differential outputs, and
  ML accuracy vs rounds for 5 ciphers.
  Output: cipher_verification_output.txt

  STEP 3 — Generate a dataset
  --------------------------------
      python -c "
      from neural_cryptanalysis.data.generator import generate_dataset, save_dataset
      from neural_cryptanalysis.utils.config   import get_cipher, DELTA_P
      cipher = get_cipher('simon32')
      X, y   = generate_dataset(cipher, rounds=4, n_samples=10000,
                                 delta_p=DELTA_P['simon32'],
                                 representation='delta')
      save_dataset(X, y, 'simon_delta_r4')
      print('Saved:', X.shape, y.shape)
      "

  Available ciphers:
      simon32, gift64, gift128, skinny64, skinny128,
      craft, pyjamask, present

  Available representations:
      delta, concat, raw, word, bitslice, statistical, joint

  STEP 4 — Train individual models
  --------------------------------
  MLP (any flat representation):
      python -m neural_cryptanalysis.experiments.train_mlp \
          --dataset neural_cryptanalysis/data/datasets/simon_delta_r4 \
          --epochs 15 --batch 256 --lr 3e-4

  CNN (requires concat or raw):
      python -m neural_cryptanalysis.experiments.train_cnn \
          --dataset neural_cryptanalysis/data/datasets/simon_concat_r4 \
          --epochs 10 --batch 128 --lr 1e-3

  Siamese (requires concat or raw):
      python -m neural_cryptanalysis.experiments.train_siamese \
          --dataset neural_cryptanalysis/data/datasets/simon_concat_r4 \
          --epochs 10 --batch 128 --lr 1e-3

  Optional flags (all scripts):
      --save path/to/model.pt     Save trained weights
      --epochs N                  Number of training epochs
      --batch  N                  Batch size
      --lr     F                  Learning rate

  STEP 5 — Run full model benchmark
  --------------------------------
      python -m neural_cryptanalysis.experiments.benchmark_models

  Trains MLP, CNN, Siamese with per-epoch logging.
  Output: model_benchmark_output.txt

  STEP 6 — Compare representations
  --------------------------------
      python -m neural_cryptanalysis.experiments.compare_representations

  Trains MLP on all 7 representations.
  Output: representation_comparison_output.txt

  STEP 7 — Run all experiments + generate plots
  --------------------------------
      python -m neural_cryptanalysis.experiments.run_all_experiments

  Runs all 3 experiments and saves 4 plots to plots/.
  Output: experiments_output.txt

  STEP 8 — Generate all datasets in bulk (optional)
  --------------------------------
      python neural_cryptanalysis/data/generate_all_datasets.py

  Optional flags:
      --cipher simon gift64       Only specific ciphers
      --repr delta concat         Only specific representations
      --n-samples 50000           Override sample count (default: 100000)
      --no-skip                   Regenerate existing datasets

================================================================
  KEY FINDINGS
================================================================

  1. Delta representation is the most effective — directly encodes
     the differential pattern C XOR C', achieving 100% accuracy
     at r=4 for simon32.

  2. Siamese network is the most parameter-efficient model (25K
     params) with 99.5% accuracy — best accuracy-per-parameter.

  3. MINE with XOR augmentation achieves 99.69% at r=4. The
     explicit differential channel [C||C'||ΔC] is critical —
     without it MINE stalls at ~55%.

  4. Cipher hardness varies significantly:
     - CRAFT becomes indistinguishable at r=4 (cliff drop)
     - Pyjamask breaks at r=3
     - SIMON32 remains distinguishable up to r=7

  5. Statistical representation with 59 rich features achieves
     99.83% — far better than the original 7-feature version
     (82.17%). Adding the raw XOR bit vector (Group 3) is the
     key improvement.

  6. All 8 ciphers pass test vector verification and avalanche
     effect checks with near-ideal bit flip counts.
